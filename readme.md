# Installation instructions
- Install 
  - Python3
  - git
  - usbmount
  - Pip3
  - Mplayer
  - Mplayer bindings: `pip install git+https://github.com/baudm/mplayer.py.git@0.7.2`
  - Flask, Flask-Cors
  - exfat-fuse, exfat-utils
- Configure
  - Make `playlists` folder read-writeable
  - Make `app.py` executable: `chmod +x app.py`
  - ...
